HTML: Hypertext Markup Language


HTML is a computer language used to structure web-based content.

HTML tags mark the beginning and end of HTML elements.

Hypertext Markup Language is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript. Wikipedia HTML is not a programming language. As its name indicates (Hypertext Markup Language), it's a markup language used for creating and structuring content on the web.

VS Code Installation

A web server launches and executes web applications in response to processing requests from web clients (i.e., browsers). This mechanism is known as CGI (Common Gateway Interface), and the web applications that run on web servers are sometimes called CGI programs. Web applications receiving processing requests use database servers as required. At this time, a processing request from a web application is passed as SQL statements to a database server, which in turn returns the execution result of the SQL statements to the web application. Then, the web application returns HTML document, dynamically generated from the received result of executing the SQL statements, to the browser, which displays the HTML document.
